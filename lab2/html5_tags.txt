Here are some of the new tags introduced in HTML5:

<header>: Represents the header section of a document or a section.
<nav>: Represents a section for navigation links.
<section>: Defines a section of a document.
<article>: Represents an independent, self-contained piece of content.
<aside>: Represents content that is tangentially related to the main content.
<main>: Specifies the main content of a document.
<footer>: Represents the footer section of a document or a section.
<figure>: Represents self-contained content, such as images, diagrams, or code snippets.
<figcaption>: Represents a caption or description for a <figure> element.
<time>: Represents a specific time or a range of time.
<mark>: Highlights or denotes a specific section of text.
<meter>: Represents a scalar measurement within a known range.
<progress>: Represents the progress of a task or process.
<details>: Represents additional details or disclosure content that can be toggled on and off.
<summary>: Represents a summary or caption for the content of a <details> element.
<datalist>: Provides a list of predefined options for an input field.
<output>: Represents the result or output of a calculation or script.
<canvas>: Provides a space for drawing graphics using JavaScript.
<video>: Embeds a video into the document.
<audio>: Embeds audio content into the document.
<source>: Specifies alternative media resources for <video> and <audio> elements.
<embed>: Embeds external content, such as plugins or multimedia, into the document.
<track>: Provides captions, subtitles, or other timed text tracks for media elements.
<datalist>: Specifies a list of pre-defined options for an input field.
<abbr>: Represents an abbreviation or acronym.
<bdi>: Isolates a span of text that might be formatted in a different direction from other text outside it.
<wbr>: Specifies a word break opportunity within a word.
